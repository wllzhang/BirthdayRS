## Project Overview

BirthdayRS is a modern birthday reminder system that provides:

- **Dual Calendar Support**: Solar (Gregorian) and Lunar (Chinese) calendar birthdays
- **Rich Date Information**: GanZhi, zodiac signs, festivals, solar terms, constellations
- **Multiple Notifications**: Email and ServerChan support
- **Flexible Deployment**: Local, Docker, and GitHub Actions
