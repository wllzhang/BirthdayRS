## 项目概述

BirthdayRS 是一个现代化的生日提醒系统，提供：

- **双历法支持**：公历（阳历）和农历（阴历）生日
- **丰富的日期信息**：干支、生肖、节日、节气、星座
- **多种通知方式**：邮件和 ServerChan 支持
- **灵活部署**：本地、Docker 和 GitHub Actions
